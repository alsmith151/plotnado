Genomic region model.

GenomicRegion ¶

Bases: BaseModel

Represents a genomic region with chromosome, start, end, and strand.

Attributes:

Name Type Description
chromosome str

Chromosome name (e.g., 'chr1')
start int

Start position (0-based)
end int

End position
strand Strand

Strand ('+' or '-')

center property ¶

Center position of the region.

length property ¶

Length of the region in base pairs.

extend(upstream=0, downstream=0) ¶

Extend the region upstream and/or downstream.

Parameters:

Name Type Description Default
upstream int

Number of base pairs to extend upstream

 0
downstream int

Number of base pairs to extend downstream

 0

Returns:

Type Description
GenomicRegion
