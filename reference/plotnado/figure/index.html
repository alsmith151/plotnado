GenomicFigure class for composing and plotting genomic tracks.

GenomicFigure ¶

Compose and plot multiple genomic tracks.

Example

fig = ( GenomicFigure() .bigwig("signal.bw", title="H3K27ac") .genes() .axis() .scalebar() ) fig.plot("chr1:1000000-2000000")
