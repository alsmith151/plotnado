# Reference Summary

This page is a lightweight index to the PlotNado API surface.

## Main entry points

- `plotnado.GenomicFigure`: compose, edit, and render figures
- `plotnado.Template`: model the YAML template format used by the CLI
- `plotnado.TemplateCompiler`: compile templates into render plans
- `plotnado.Theme`: define figure-wide defaults for styling and spacing
