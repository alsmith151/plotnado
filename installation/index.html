Installation¶

Requirements¶

  • Python 3.12+
  • pip

Install from PyPI¶

pip install plotnado

Development install¶

git clone https://github.com/alsmith151/plotnado
cd plotnado
pip install -e .[dev,docs]
pre-commit install

Verify installation¶

python -c "import plotnado; print(plotnado.__version__)"
plotnado track-options bigwig

Build docs locally¶

mkdocs build --strict
mkdocs serve
